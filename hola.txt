## Modelos de Datos

- **Modelo Relacional:**  
  Basado en tablas con relaciones entre ellas usando claves primarias y foráneas.

- **Modelo de Documentos:**  
  Datos almacenados en documentos tipo JSON o BSON.

- **Modelo de Grafos:**  
  Datos representados como nodos y relaciones, útil para redes sociales y recomendaciones.

- **Modelo Jerárquico:**  
  Datos organizados en estructura tipo árbol.



  ## Lenguajes para Bases de Datos

- **SQL (Structured Query Language):**  
  Lenguaje estándar para bases de datos relacionales para realizar consultas, inserciones, actualizaciones y borrados.

- **NoSQL Query Languages:**  
  Lenguajes específicos o APIs para bases NoSQL (por ejemplo, MongoDB Query Language).git
